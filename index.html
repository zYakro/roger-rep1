Tabla de Etiquetas HTML
<html> Define el documento como una página web en HTML.
<head> Contiene metadatos sobre el documento como el título y enlaces a archivos externos.
<title> Define el título del documento, que se muestra en la pestaña del navegador.
<body> Contiene el contenido visible de la página web.
<h1> - <h6> Define los encabezados, siendo <h1> el más importante y <h6> el menos importante.
<p> Define un párrafo de texto.
<a> Define un enlace a otra página o recurso.
<img> Se utiliza para insertar imágenes en una página web.
<ul> Define una lista desordenada (con viñetas).
<li> Define un ítem dentro de una lista (se usa dentro de <ul> o <ol>).
